**********************************JEU DU BAR A COCKTAILS**********************************
Instructions :
Vous avez le choix d'incarner soit le barman ou le client pour jouer au jeu
Si vous êtes client, plusieurs options s'offrent alors à vous :

- Commander des boissons et/ou des cocktails dans la carte du bar
- Créer votre propre cocktail avec les boissons qui vous font plaisir

Et si vous êtes barman, vous pouvez réaliser les actions suivantes :

- Attendre la prise de commande d'un client aléatoire
- Gérer le stock de boissons (remplir le stock, changer les prix)
- Afficher votre recette du jour

Vous pouvez également choisir d'incarner les deux personnages à la fois pour obtenir une 
vision plus gobale du bar !
